### general-info

This is udacity capstone project for [Udacity Nanodegree program](https://www.udacity.com/course/blockchain-developer-nanodegree--nd1309).

This project aim to digitalize property sell/purchase using ethereum platform. Property will be listed as Non Fungible Token (ERC-721) and can traded on [OpenSea Store](https://opensea.io/). This project also implement [Zero Knowledge Proof using Zk-SNARK](https://z.cash/technology/zksnarks/). To implement Zk-SNARK we will use Zo-krates a toolkit to implement zero knowledge proof on Etheream platform.

### installation
A step by step series of examples that tell you have to get a development env running

Clone this repository:

```bash
git clone https://github.com/Tarunshrma/Blockchain-Capstone.git
```

Change directory to FlightSurety folder and install all requisite npm packages (as listed in package.json):

```bash
cd BlockchainCapstone
npm install
```

Compile the smart contracts
```bash
truffle compile
```

### unit-test
Open the terminal and launch the local instance of blockchain to test your smart contracts. 
```bash
truffle develop
```
This command will launch the local blockchain with 10 test accounts and private keys. You can use those accounts to test and execute the smart contracts locally without deploying it to TestNet or MainNet. 

Run Unit Test Cases
```bash
test
```
